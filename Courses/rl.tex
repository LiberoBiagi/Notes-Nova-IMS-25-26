\section{Reinforcement learning}
\subsection{Introduction}

We have to balance exploration and exploitation in reinforcement learning. Reinforcement learning is a branch of machine learning where an agent tries to make decisions basing its choice on experience and interactions with the environment. It receives a numerical reward signal that indicates how well it is doing and gradually improves its behavior via trial and error. 

\vspace{10pt}


The most important elements are 
\begin{itemize}
    \item Agent
    \item Action
    \item Environment
    \item Reward
\end{itemize}

We want to maximize long term goals. We get an input that we translate into an environment and with our agent we look for a certain output.
